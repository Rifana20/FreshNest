# FRESHNEST - Food Expiry Tracker + AI Recipe Suggestions

## Overview

Food Expiry Tracker is a web application that helps users manage food inventory, monitor expiry dates, reduce food waste, and generate AI-powered recipe suggestions using available ingredients.

---

## Problem

Many households waste food because they forget expiry dates or do not know how to utilize available ingredients effectively.

This project aims to reduce food waste through smart inventory management and AI-generated recipe recommendations.

---

## Planned Features

### User Authentication

* Register
* Login
* Secure access

### Food Inventory Management

* Add food items
* Edit food items
* Delete food items
* View food inventory

### Expiry Tracking

* Fresh items
* Expiring soon items
* Expired items

### AI Recipe Suggestions

* Generate recipes from available ingredients
* Prioritize ingredients nearing expiry
